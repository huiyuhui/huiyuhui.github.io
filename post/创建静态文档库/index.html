使用Hugo创建静态文档库

#我的静态文档库

  1. 在本地任意目录下执行命令Hugo new site MyDoc创建本地文档库
  2. 执行命令tree查看目录结构，如下所示：
    • Archetypes：存放default.md，头文件格式
    • Content：存放博客文章（markdown文件）
    • data：存放自定义模板，导入的toml文件或json，yaml文件
    • layouts：存放网站的模板文件
    • static：存放图片，css等静态资源
    • config.toml：网站的配置文件
  3. 进入git bash执行以下命令下载主题
    cd d:
    cd Huiyuhui/MyDoc
    Git clone https://github.com/h-enk/doks.git themes/doks
  4. 执行以下命令设置主题
    echo ‘theme=”docsy”’ » config.toml
  5. 执行以下命令测试网站是否成功
    hugo serve 运行成功后通过浏览器访问config文件中的网址即可
