Configure the Static Pages to Github

1.Visit github official website to sign up?
2.Create your Personal repository.
3.Run the command hugo –theme=techdoc –baseURL=“https://huiyuhui.github.io/" to generate the pages.
4.Enter the folder “public”, and run the following command to push the local files to your repository.
git init
** remote add origin https://github.com/huiyuhui/huiyuhui.github.io.git**
**git add -A**
**git commit -m “first commit”**
**git push -u origin master**
