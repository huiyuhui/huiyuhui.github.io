Environmental Preparation

Installing Hugo

1.Visit Hugo official website to download the package, such as hugo_extended_0.80.0_Windows-64bit.zip.
2.Unzip hugo package to the folder D:hugo.
3.Open the Control Panel, and select System > Advanced System Settings.
4.Setting the environment variables for Windows system.
5.Run the command Hugo version in CMD window to check whether the Hugo is installed successfully.

Installing GIT

1.Visit Git official website to download the Git package.
2.Click Next until the Git is installed completely.
